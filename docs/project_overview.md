# Project Overview

## Cloud-Native Payment Analytics Platform

### Objective
Build an end-to-end AWS data pipeline that ingests, cleans, stores, and analyzes payment transaction data.

### Architecture
Transaction Data → Amazon S3 → AWS Lambda → Amazon RDS MySQL → Power BI

### Technologies
- AWS S3
- AWS Lambda
- Amazon RDS MySQL
- IAM
- CloudWatch
- Python (Pandas, Boto3)
- SQL
- Power BI

### Key Features
- Automated data ingestion
- Data cleaning and validation
- Transaction monitoring
- Interactive dashboards
